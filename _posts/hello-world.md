#### 常用命令：

| hexo clean hexo g hexo d | 上传                                                         |
| :----------------------- | ------------------------------------------------------------ |
| hexo g                   | 生成静态网页                                                 |
| hexo s                   | 打开本地服务器 http://localhost:4000/                        |
| hexo d                   | 部署到远端服务器（github）git不更新就删除.deploy文件，重新部署 |
| hexo clean               | 清除缓存（public文件夹下）                                   |
| hexo new “name”          | 创建新文件名                                                 |

| git pull            | 更新                       |
